)
inlineconstexprnoexcept

sets the given bits in the register and overrides the leftover bits with zeros
equivalent to C register access -> reg = value1 | value2 | ~...

Usage:

MT::Universal::Register<&HWRegister> reg1;
reg1.override(BITS8::B0 | BITS8::B1)
Template Parameters
BITSbits or bitmask that should be set into the defined register -> template specialization for Enum types (BITS8,BITS16,BITS32)
